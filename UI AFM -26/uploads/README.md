# American Football GM Simulation MVP

This project is a web-based American football GM simulation game.

The MVP target is a fictional, compressed league with NFL-inspired structure, full GM roster/contract control, playbook-driven drive-level game simulation, defensive stat tracking, autosaves, and a 6-team playoff format.

## Stage Progress

- Stage 1: Game design rules - drafted in `docs/stage-1-game-design-rules.md`
- Stage 2: Data layer - implemented in `docs/stage-2-data-layer.md`
- Stage 3: Simulation engine - awaiting clearance
- Stage 4+: Player-facing web MVP - awaiting clearance
